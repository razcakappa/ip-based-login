== Description ==

IP Based Login allows you to directly login from an allowed IP. So if you want to allow someone to login but you do not want to share the login details just add their IP using IP Based Login.

Features in IP Based Login 1.3.3 include:

* Create IP ranges
* Choose the username accessible when accessed by the IP existing in provided range
* Delete IP ranges
* Enable/Disable IP ranges
* Completely FREE
* Licensed under GNU GPL version 3
* Does not affect when accessed from any other IPs not existing in any ranges
* Safe & Secure
* No passwords saved


== Installation ==

Upload the IP Based Login plugin to your blog, Activate it.
That's it. You're done!
